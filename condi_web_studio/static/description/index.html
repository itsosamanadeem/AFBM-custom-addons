Overview

This module enables you to restrict view modifications via the Enterprise Studio module.

Users can no longer alter or edit views through the breadcrumb menu located at the top or by using the same menu in list view to add custom fields.

Features

Able to Access the Studio with ACL Access.

It Indicates that authorized individuals or roles have the ability to modify and customize your software within the boundaries set by your access control policies. It ensures that only those with the appropriate permissions can make changes while maintaining security and control over your system.

Disable the Studio Menu.

This action would disablee the menu, preventing users from accessing or using the Studio functionality via the breadcrumb navigation. It's a straightforward request to restrict access to Studio features via this specific menu option.

Turn off Studio View Editability in Listview.

Thiis modification will likely require going through ACL to enable/disable this feature..
